# Adobe Web SDK + AEP + CJA Mock Implementation

## Overview
This project demonstrates a mock end-to-end Adobe Experience Platform
implementation using Web SDK, event-driven data layer, XDM schema design,
and Customer Journey Analytics concepts.

⚠️ This is a NON-PRODUCTION, MOCK implementation built for learning and
interview demonstration purposes only.

## Architecture
Browser → Data Layer → Web SDK (Mock) → AEP → Adobe Analytics → CJA

## Key Features
- Event-driven digital data layer
- Web SDK payload simulation
- XDM Experience Event schema
- Adobe Launch rule design
- Debugging via Network tab
- CJA funnel modeling

## Events Tracked
- Product View
- Add to Cart
- Checkout Start

## Debugging
- Chrome DevTools Network tab
- Payload validation
- Event de-duplication checks

## Why Web SDK
- Unified data collection
- Supports AEP, Analytics, Target
- Schema-based tracking
